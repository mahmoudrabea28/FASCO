Premium Design: A sophisticated aesthetic inspired by modern fashion trends.

Persistent Mini-Cart: A slide-over cart that uses LocalStorage to keep your items saved even after a page refresh.

Real-time Cart Management: Add, remove, and update product quantities directly within a dynamic array.

Fully Responsive: Optimized for a flawless experience across mobile, tablet, and desktop devices.

Interactive Instagram Feed: A curated gallery layout showcasing social media integration.

Complete User Flow: Includes functional pages for Home, Product Details, Sign In, and Sign Up.

🛠️ Tech Stack
HTML5 & CSS3: Semantic structure and custom styling.

Tailwind CSS: For rapid, utility-first responsive design.

JavaScript (ES6+): Powers the cart logic, DOM manipulation, and data persistence.

Google Fonts: Utilizing Playfair Display for headings and Poppins for body text to maintain a luxury brand feel.
